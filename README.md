# LangGraph React Course

A Python project for learning LangGraph with React integration.

## Description

This is a course project that demonstrates how to use LangGraph with React.

## Installation

Using uv:

```bash
uv sync
```

## Usage

```bash
uv run <your-script.py>
```

## Dependencies

- langchain
- langgraph
- langchain-openai
- langchainhub
- black
- isort
- python-dotenv
- langchain-tavily
